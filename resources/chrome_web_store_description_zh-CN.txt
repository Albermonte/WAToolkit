为 WhatsApp™ Web 用户设计的一组轻载多功能工具。

现在，当您在使用台式电脑时，再也不会错过 WhatsApp 的通知了！此扩展功能可让您在没有打开任何 WhatsApp 选项卡或 Chrome 窗口时也能收到 WhatsApp。这样您便可在使用电脑的同时不再错过任何信息了。

————————————————————
特点
————————————————————

★ 即时桌面通知：Chrome 一直以后台模式运行，跟踪并即时向您推送 WhatsApp 通知。您可以点击通知，直接进入相关对话。

★ 在工具栏添加一个 WhatsApp 便捷按钮：还可显示有几条未读信息。您可以将鼠标置于图标之上，将出现提示信息显示您的未读信息的总数。此外，当您的手机出现连接问题时，图标将变成橙色。您还可以隐藏按钮，只需右击图标选择"隐藏按钮"。

★ 确保您每次只打开一个 WhatsApp Web 窗口：这样不管在什么情况下，您都可以找到正确的选项卡，此外，无需处理 WhatsApp 退出确认信息。

★ 超级轻载且快速：以性能为导向的编码。

————————————————————
常见问题
————————————————————

• 如何使用？首先，安装此扩展功能，然后点击 Chrome 工具栏上带 "T" 图标的按钮。如果您未登录 WhatsApp Web，您必须按照指示，用您的手机扫描二维码。请记得勾选“保持登录”复选框，以确保中途不会自动关闭，否则您可能会错过通知。

• 为什么我看不到任何通知？如果工具栏上的"T" 图标显示为橙色，那么您的手机可能出现了连接问题。如果您还是看不到 WhatsApp 通知，请参考官方支持文章，解决您的问题：http://www.whatsapp.com/faq/web/28080004

MOTIVATION
————————————————————

As all the software I make in my free time, I created WAToolkit because I wanted the functionality it offers for myself. My first need was being able to watch all WhatsApp notifications on my desktop computer, even without a WhatsApp tab opened (or any Chrome window at all), and that was its first feature. As the extension has raised some interest, I'll be maintaining WAToolkit and new features will probably be added in the future. Be sure to report bugs or suggest new ideas by clicking on the SUPPORT tab.

PERMISSIONS
————————————————————

The message shown by Google Chrome for the requested permissions is far from the real needs of the extension, but the permissions are technically required for the following reasons:

• Read and change your data on all whatsapp.com sites: Required to create the background page that holds an active WhatsApp session.

• Read your browsing history: Required to detect whether an active WhatsApp session exists among the tabs you are browsing, avoiding conflicts with the background page.
